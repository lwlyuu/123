# 123
Программный комплекс на Python для эффективного хранения и анализа пространственной информации. Проект демонстрирует мощь R-tree индексов при работе с большими данными, обеспечивая мгновенный радиальный поиск. Включает интерактивный GUI с тепловыми картами, маршрутизацией и визуализацией структуры дерева (MBR).
Основные возможности:
1. Высокая производительность: эффективная обработка и визуализация до 2 000 000 объектов в реальном времени.
2. Пространственный индекс: реализация динамического R-tree с визуализацией ограничивающих прямоугольников (MBR).
3. Интерактивный анализ: поиск ближайших объектов кликом по карте, фильтрация по типам и рейтингу.
4. Тепловые карты: генерация карт плотности застройки с использованием Гауссова размытия для выявления кластеров данных.
5. Маршрутизация: инструмент для построения путей и мгновенного расчета дистанции между точками.
6. История запросов: система логирования и быстрого возврата к результатам предыдущих поисков.
